# g-n-rateur-Cem
Générateur Pédagogique CEM: Plateforme intelligente dédiée aux professeurs de français en Algérie. Elle permet de générer des textes, situations d'intégration et grilles d'évaluation conformes au programme. Inclut un assistant IA (Gemini) pour la correction automatique des productions écrites par OCR. أداة ذكية للأستاذ الجزائري.
